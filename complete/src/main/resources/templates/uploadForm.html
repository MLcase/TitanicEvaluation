Titanic Evaluation

Upload (.csv)

Files with one of the following formats can be evaluated by this app:

Format One:

Surname, Forename, Age, Class (first character must be class number, otherwise part of crew), Died (true if this cell has any content)

Format Two:

Class (single digit), Survived ("1"=true; "0"=false), Name, Sex ("male" or "female"), Age

Any other lines or cells beyond this format will be ignored. One file should only contain one kind of format.
